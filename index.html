Sobre Mim

Me chamo Hábia Coutinho, Atualmente estudo desenvolvimento web, com foco em front-end, e áreas relacionadas à Análise e Desenvolvimento de Sistemas por meio de cursos online.

Meu primeiro contato com programação aconteceu em 2024, quando participei de um curso de Desenvolvimento de Sistemas no Senac. Esse foi meu ponto de partida no universo da tecnologia.

A partir dessa experiência, passei a aprofundar meus estudos de forma independente, utilizando plataformas online como Coursera e conteúdos complementares do YouTube. Nesse processo, concluí cursos que ajudaram a fortalecer minha base técnica.

No Senac, tive contato com lógica de programação, algoritmos, desenvolvimento web com HTML, CSS e JavaScript, introdução a Python e conceitos de banco de dados com MySQL.

Também tive introdução a práticas como CRUD, testes automatizados e integração entre front-end e back-end, o que contribuiu para minha compreensão do funcionamento de aplicações completas.

Tenho interesse em desenvolvimento Front-End e sigo estudando tecnologias como JavaScript e Python, com o objetivo de evoluir futuramente para desenvolvimento Back-End e Full Stack.

Continuo construindo minha trajetória na área de tecnologia por meio de prática constante, estudos e desenvolvimento de projetos pessoais.

Meu foco é criar projetos responsivos e acessíveis, sempre buscando melhorar a experiência do usuário.
